# Kasidech Chumthong | QA Engineer & Developer Portfolio

A premium, high-performance portfolio website inspired by the clean, minimalist aesthetic of **Linear.app**. Built with **React** and **Vite**, featuring advanced interactive elements and a focus on software quality.

🚀 **Live Demo**: [https://portfolio-kasidech.vercel.app/](https://portfolio-kasidech.vercel.app/)

## ✨ Key Features

- 🖥️ **Linear Design System**: Sleek dark mode, glassmorphism, and precise typography.
- ⌘ **Command Palette**: Quick navigation and actions using `Cmd + K` (or `Ctrl + K`).
- 🏗️ **Premium Project Showcase**: Interactive project cards with deep-dive detail modals and status verification.
- 🌓 **Dynamic Theming**: Seamless switching between Dark and Light modes.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide displays.
- 🛠️ **QA Identity**: Integrated automation metrics and verified certification badges.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Linear Design System)
- **Icons**: Simple Icons & Lucide-inspired SVGs
- **Deployment**: Vercel

## 📂 Project Structure

```text
src/
├── components/     # Reusable UI components (Navbar, Hero, Projects, etc.)
├── constants/      # Content and configuration (index.js)
├── hooks/          # Custom React hooks (useTheme, useReveal, etc.)
├── assets/         # Static assets (Logos, Images)
└── style.css       # Core design system and global styles
```

## ⌨️ Command Palette Actions

- **Navigation**: Quickly jump to About, Skills, Projects, or Experience.
- **Search**: Find specific projects by name or category.
- **Theme**: Toggle between Light and Dark modes instantly.
